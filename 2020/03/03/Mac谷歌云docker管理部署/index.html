Mac谷歌云docker管理部署

3月 3, 2020 阅读量

安装docker

curl -sSL https://get.docker.com/ | sh

部署docker管理界面

sudo docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock \
--name ui --restart=always lihaixin/portainer:me

部署有提示出错则

1、普通用户权限问题

sudo service docker restart

2、创建新Docker容器时出现“The container name “/xxx” is already in use by container xxxxxxxxxxx…”问题的解决办法

查看所有容器移除及创建

docker ps -a
docker rm e3274a72e8d6（容器名 CONTAINER ID）
docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock \
--name ui --restart=always lihaixin/portainer:me

3、ssh链接
ssh 公开密钥名称@ip地址
